What is this?

A project by Josh Krauth-Harding

Project Solis is a project in appreciation of the primarily African American women who powered the 1960's space race for a country that widely neglected them.


The prompt for this project was to create an online tool that would improve their everyday lives had they had the internet at the time. Featuring some rudementary tools like a chat feature and a basic online wiki, Solis attempts to be a companion to those women.
